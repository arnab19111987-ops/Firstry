# Install FirstTry VS Code Extension (Offline .vsix)

1. Download the latest `.vsix` from the Releases page (look for `firsttry-extension-*.vsix`).
2. Open VS Code.
3. Go to:
   - Extensions sidebar (Ctrl+Shift+X)
   - Click the `...` menu (top right of the sidebar)
   - Choose `Install from VSIX...`
4. Select the downloaded file.
5. Reload VS Code when prompted.

## What you get
- A "FirstTry" panel that runs all local checks before commit.
- One-click “Run FirstTry Now” command.

## Common issues
- VS Code may block unknown publishers the first time. Click "Install Anyway".
- If you already have a dev build installed, uninstall it first.
